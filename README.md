## 🔧 Работа с Ruff

Ruff используется для автоматического форматирования и проверки кода.

| Команда | Что делает |
|---------|------------|
| `ruff check` | Проверяет код на ошибки и стиль |
| `ruff check --fix` | Автоматически исправляет ошибки |
| `ruff format` | Форматирует код по стандарту |
